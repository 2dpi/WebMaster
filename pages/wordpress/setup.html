Installing Wordpress over a network using Webmatrix

Issue: Once the port is open and you have shared the site over the network the site will break because you have changed the site urls in Webmatirx from localhost:xxxx* to pc-name*:xxxx

*xxxx = port no
*pc-name: = the-bridge:

Fix: Before you change the site url in Webmatrix, first log in to the backend of Wordpress and click settings, general, change: 1) Wordpress Address (URL): pc-name:xxxx and 2) Site Address (URL): pc-name*:xxxx to what you are going to set it in Webmatrix, then open the port and share over the network

Wordpress Admin Login: http://pc-name:xxxx/wp-admin
