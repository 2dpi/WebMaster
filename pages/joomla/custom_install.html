Joomla Custom Installation Files

1) Install Joomla latest version
2) Install W3 Framework (customised template) First zip before installing
5) Install custom modules and plugins as appropriate
Plugins:

  • plg_content_shortcodegallery (customisable gallery)
  • plg_system_paramboost (extends menu/module & component parameters and also allows additional attributes for content)
  • plg_content_youtubegallery (to add a youtube gallery)
  • plg_content_youtubeembed (to embed a youtube video - allowing multiple on a page)
  • plg_user_fileupload (to allow a user on registration to upload a file)
  • plg_user_printprofile (allows the system administrator to print a registered user profile)
  • plg_user_profilextender (extends user profile attributes)
Modules:
  • mod_customcode (passes php)
  • mod_menu (reponsive menu) Set up required

3) Make the www framework template the default
4) Set template parameters
JS Includes & Excludes add: html5.js, respond.min.js, selectivizr-min.js
CSS Includes & Excludes add: alpha, buttons, forms, grid, icons, layout, normalisze, tables, xtra and any others as appropriate
Layout Contnet Area:
Default Layout: layout_cck - build your own
Custom Layout: normally layout_2_col - component section (joomla, k2, seblod, fabrik etc and an aside column (modules only)) Assign layout to appropriate menu items. To turn the aside column off select menu items where you want hide the aside column. To hide the component select menu items where you want to hide the component. For example the Home Page may only be made up of Modules and a Slider. To hide the component select menu items as appropriate. To hide the aside column select menu items as appropriate
Custom Fonts: Add google fonts
Copyright: set
